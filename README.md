# AutoThemeSwitcherNG

AutoThemeSwitcherNG is a Visual Studio extension that automatically synchronizes your Visual Studio 2026 theme with your Windows system theme (Light/Dark mode).

## Features
- **Automatic Synchronization**: Monitors Windows Registry for system theme changes and updates Visual Studio theme instantly.
- **Customizable Themes**: Choose which Visual Studio themes to use for Light and Dark modes via Tools -> Options.
- **Lightweight**: Uses efficient registry monitoring instead of polling.

## Usage
1. Install the extension.
2. The extension works automatically in the background.
3. To configure specific themes:
   - Go to **Tools** -> **Options**.
   - Navigate to **Auto Theme Switcher NG** -> **General**.
   - Set your preferred **Light Theme Name** (default: "Light") and **Dark Theme Name** (default: "Dark").
